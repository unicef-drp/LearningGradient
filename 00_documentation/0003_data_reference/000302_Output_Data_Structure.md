# Output Data Structure

This document describes the structure of the two output files generated by `02_scripts/0202_transform.R`.

---

## Output Files

| File | Description | Location |
|------|-------------|----------|
| `030101_output_grade.csv` | Grade-level proficiency data (grades 1-8) | `03_output/0301_tables/` |
| `030102_output_grade_band.csv` | Grade-band proficiency data (3 bands) | `03_output/0301_tables/` |

---

## 1. 030101_output_grade.csv

Individual grade-level data with proficiency rates for grades 1-8.

### Columns

| Column | Type | Description |
|--------|------|-------------|
| `iso3` | string | ISO 3166-1 alpha-3 country code |
| `country_name` | string | Country name (from UNICEF) |
| `disaggregation` | string | Disaggregation type (see below) |
| `category` | string | Disaggregation category (see below) |
| `subject` | string | Subject: `reading` or `numeracy` |
| `grade` | integer | Grade level (1-8) |
| `is_africa` | boolean | Whether country is in Africa (UNICEF region) |
| `income_level` | string | World Bank income classification |
| `income_level_code` | string | World Bank income code (LIC, LMC, UMC, HIC) |
| `Region_Code` | string | UNICEF region code |
| `Region` | string | UNICEF region name |
| `n_unweighted` | integer | Sample size (unweighted count) |
| `pct_weighted` | numeric | Proficiency rate - weighted (%) |
| `n_weighted` | integer | Effective sample size for weighted calculation |
| `pct_unweighted` | numeric | Proficiency rate - unweighted (%) |
| `pct_weighted_loess_span1` | numeric | LOESS-smoothed proficiency rate (span=1) |

### Disaggregation Values

| Value | Categories Included |
|-------|---------------------|
| `All` | All (total population) |
| `Sex` | Female, Male |
| `Area` | Urban, Rural |
| `Wealth` | Poorest, Second, Middle, Fourth, Richest |

### Category Values

| Value | Description |
|-------|-------------|
| `All` | Total population |
| `Female` | Female students |
| `Male` | Male students |
| `Urban` | Urban residence |
| `Rural` | Rural residence |
| `Poorest` | Wealth quintile 1 (poorest 20%) |
| `Second` | Wealth quintile 2 |
| `Middle` | Wealth quintile 3 |
| `Fourth` | Wealth quintile 4 |
| `Richest` | Wealth quintile 5 (richest 20%) |

## 2. 030102_output_grade_band.csv

Aggregated grade-band data with proficiency rates for 3 education stages.

### Columns

| Column | Type | Description |
|--------|------|-------------|
| `iso3` | string | ISO 3166-1 alpha-3 country code |
| `country_name` | string | Country name (from UNICEF) |
| `disaggregation` | string | Disaggregation type (same as grade file) |
| `category` | string | Disaggregation category (same as grade file) |
| `subject` | string | Subject: `reading` or `numeracy` |
| `grade_band` | factor | Grade band (see below) |
| `is_africa` | boolean | Whether country is in Africa (UNICEF region) |
| `income_level` | string | World Bank income classification |
| `income_level_code` | string | World Bank income code (LIC, LMC, UMC, HIC) |
| `Region_Code` | string | UNICEF region code |
| `Region` | string | UNICEF region name |
| `n_unweighted` | integer | Sample size (unweighted count) |
| `pct_weighted` | numeric | Proficiency rate - weighted (%) |
| `n_weighted` | integer | Effective sample size for weighted calculation |
| `pct_unweighted` | numeric | Proficiency rate - unweighted (%) |
| `pct_weighted_loess_span1` | numeric | LOESS-smoothed proficiency rate (span=1) |

### Grade Band Values

| Value | Grades Included | Numeric Position (for LOESS) |
|-------|-----------------|------------------------------|
| `Early Primary (1-3)` | Grades 1-3 | 2 (midpoint) |
| `End of Primary (4-6)` | Grades 4-6 | 5 (midpoint) |
| `Lower Secondary (7-9)` | Grades 7-9 | 8 (midpoint) |

## Enrichment Data Sources

### UNICEF Countries (country names and regions)

- **Source**: UNICEF Country-and-Region-Metadata API (fetched live by `0201_load.R`)
- **Provides**: `country_name`, `Region_Code`, `Region`
- **Example regions**: South Asia, Eastern and Southern Africa, West and Central Africa, etc.

### World Bank Income Classification

- **Source**: World Bank API (`https://api.worldbank.org/v2/country`)
- **Provides**: `income_level`, `income_level_code`
- **Codes**: 
  - `LIC` = Low income
  - `LMC` = Lower middle income
  - `UMC` = Upper middle income
  - `HIC` = High income

### African Countries

- **Source**: Derived from the UNICEF Country-and-Region-Metadata API; countries where `Region_Code %in% c("UNICEF_ESARO", "UNICEF_WCARO")` (Eastern/Southern Africa and West/Central Africa) are merged to a single `UNICEF_AFRICA` group
- **Used for**: `is_africa` boolean flag

---

## LOESS Smoothing

The `pct_weighted_loess_span1` column contains LOESS-smoothed proficiency rates:

- **Method**: Local polynomial regression (LOESS) with `span = 1`
- **Grouping**: Calculated separately for each (iso3, category, subject) combination
- **Purpose**: Provides smoothed trend lines for visualization

### Notes on LOESS

- For **grade data** (8 points per group): LOESS provides meaningful smoothing
- For **grade band data** (3 points per group): LOESS with span=1 passes through all points (minimal smoothing effect)
- When fewer than 3 unique x-values exist, linear interpolation is used as fallback
